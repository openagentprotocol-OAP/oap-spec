# Charter Conformance Tests

Charter tests verify that the User Sovereignty Charter of [RFC 0016](../../rfcs/RFC-0016-user-sovereignty-charter.md) is observably honored at runtime, not merely declared in prose. These tests intentionally cross technical and ethical boundaries because the charter is what binds them together. A test that passes the schema and behavior suites but fails the charter suite indicates an implementation that is technically conformant but ethically deficient, which under section 7 of RFC 0016 forfeits the right to claim OAP conformance.
