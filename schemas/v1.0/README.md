# JSON Schemas

Normative JSON Schema 2020-12 documents for the Open Agent Protocol, version 1.0.

| File | Subject |
|------|---------|
| `oap-manifest.schema.json` | Tool Manifest |
| `oap-action.schema.json` | Action descriptor |
| `oap-request-envelope.schema.json` | Invocation request envelope |
| `oap-response-envelope.schema.json` | Invocation response envelope |
| `oap-receipt.schema.json` | Receipt of any type |
| `oap-decision-record.schema.json` | Policy Decision Record |
| `oap-ccc.schema.json` | Confidentiality and Compliance Context |
| `oap-subscription.schema.json` | Subscription state |
| `oap-wallet-statement.schema.json` | Signed Wallet statement |
| `oap-incident.schema.json` | Incident report |
| `oap-deletion-receipt.schema.json` | Deletion Receipt |
| `oap-attestation.schema.json` | Outcome Attestation |

Schemas are versioned independently of the specification text. Conformance is asserted against this directory in combination with `spec/v1.0/OAP-CORE-1.0.md`.
